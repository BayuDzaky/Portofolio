About

Game adalah permainan yang diprogram pada perangkat yang dapat dijalankan secara offline maupun online. Berdasarkan cara bermainnya, game dapat dijalankan pada perangkat yang tidak terhubung ke internet (offline) dan harus terhubung ke internet (online) untuk dapat mengakses server terpusat. Oleh karena itu, pengguna tidak dapat memainkan game online jika perangkat mereka tidak terhubung ke internet.

Game juga dapat dikelompokkan menjadi beberapa bagian. Berdasarkan alat atau platform yang digunakan, jenis permainan salah satunya, yaitu:

  • Mobile Games.

    Mobile games merupakan game yang digandrungi oleh semua orang. Pasalnya, game mudah dipasang dan dimainkan di perangkat seluler. Mobile games ini juga terbagi menjadi dua lagi yaitu game yang bisa dimainkan secara offline atau online. Contoh game yang populer adalah mobile legend, free fire, pubg dan lain sebagainya.
